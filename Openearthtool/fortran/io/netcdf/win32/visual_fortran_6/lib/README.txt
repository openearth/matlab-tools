This directory contains prebuild libraries:
- for windows 32bit in the win32 folder
  \win32\all\hdf5 from ftp://ftp.unidata.ucar.edu/pub/netcdf/contrib/win32/5-181-win-vs2005.zip
  \win32\all\szip from ftp://ftp.unidata.ucar.edu/pub/netcdf/contrib/win32/szip21-vs6-enc.zip
  \win32\all\zlib from ftp://ftp.unidata.ucar.edu/pub/netcdf/contrib/win32/zlib123-vs2005.zip
  \netcdff90      from http://www.unidata.ucar.edu/downloads/netcdf/netcdf-4_0_1/index.jsp (slightly different)
  \win32\vs\f2c   from ftp://ftp.unidata.ucar.edu/pub/netcdf/contrib/win32/win32_vs_f2c_dll_4.0.1.zip
  \win32\vs\pg    from ftp://ftp.unidata.ucar.edu/pub/netcdf/contrib/win32/win32_vs_PGI_dll_4.0.1.zip
