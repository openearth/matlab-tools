The IMSL binaries have some issues with error handling through, 
compiling dzanly from scratch using the old IMSL fortran routines did work.
These routines are not shipped with the SWANmyud code due to copyright issues.
Please compile SWANmud with your onw IMSL libraries (either binary or *.for).
