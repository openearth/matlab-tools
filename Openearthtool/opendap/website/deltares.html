This is the OPeNDAP server of Deltares. We currently have 2 different implementations of OPeNDAP running.

  • THREDDS server
  • HYRAX server

These two implementations interface the same fileserver. This OPeNDAP server is maintained by the OpenEarth initiative, for more information please see openearth.nl, openearth.eu.

OpenEarth, and such this OPeNDAP server, are sponsored jointly by:

  • EU MICORE program
  • The Building with Nature research program administratered by Ecoshape

For more information on OPeNDAP technology please refer to OPeNDAP.org (HYRAX) or Unidata (THREDDS).
