Skillbed

Introduction

Welcome to the automated Skillbed Interface. The Skillbed monitors the qualitative performence of models by automatically running a selection of models with the latest model version from a Subversion repository. The results are analyzed using Matlab scripts that provide visual and statistical comparisons with measurements and past results. The results of the analysis is collected on several PDF reports, which are sent to the development team.

This interface enables users to start additional runs of the skillbed, possibly with selected binaries, tests and reports. Please walk through the steps on the right of this frame by dragging your mouse to the left. Follow the instructions displayed for each step.

Configuration

Please select from the list below the types of runs should be executed. Currently, two types exists: custom and default. The custom type is the full model as constructed by the modeller. The default type omits all non-default settings and runs the model as if there was nog specific configuration.
