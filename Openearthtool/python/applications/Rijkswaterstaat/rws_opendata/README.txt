Rijkswaterstaat opendata
=======================

Overview
--------
This package extracts data from

http://www.rws.nl/rws/opendata/meetdata/meetdata.zip

and

http://live.waterbase.nl/metis/cgi-bin/mivd.pl?action=value&format=xml&lang=nl&order=code&type=loc

To get the code please subscribe to

http://oss.deltares.nl

And download the OpenEarthTools

Testing
-------

Tests should be placed in the "tests" directory, or subdirectories of that.
The sample test included works with unittest.
