The Nesting software is based on the Fortran programs I wrote over 15 years ago. The methodology is essentially the same. However, compared to the previous versions I also support:

 

1.      Nesting of parallel velocity components,

2.      Nesting of Riemann boundaries,

3.      Nesting of Neuman (water level gradients) boundaries,

4.      Nesting of SIMONA (Rijkswaterstaat) models. You can nest a SIMONA model into another SIMONA model or into a Delft3D-Flow model. The same applies to Delft3D-Flow models which can be nested into another Delft3D-Flow model or a SIMONA model. Filenames or used to identify whether it is a SIMONA or Delft3D-Flow file. Use Default names.

5.      A GUI is used to select files and add additional nesting information.

 

I am still testing the new functionality so take care when nesting parallel velocity, Riemann or Neuman boundaries.

 

I rely heavy on functions already written by colleagues. Credits should go to Bert Jagers for his extensive set of tools for reading and writing files.
