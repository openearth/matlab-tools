                  6 % triangulates and quadrangulates a mesh 
                  7 % into triangles and quadrangles. It also gives the mappers
                  8 % array of the traingles to the quadrangles and v.v.
                  9 % This is faster than using DELAUNAY and QUAT and 
                 10 % finding out the mapping afterwards. In TRIQUAT
                 11 % the triangulation and quadrangulation are performed
                 12 % in a structured way so the mapping is made without 
                 13 % extra comparisons.
                 14 % 
                 15 % It does the same as the following 3 actions:
                 16 %
                 17 %    delaunay(x,y)
                 18 %    quat(x,y)
                 19 %    tri2quat(tri,quat)
                 20 %
                 21 % Do note that ALL TRIANGLES ARE ORIENTED IN THE SAME WAY,
                 22 % whereas DELAUNAY has a random distribution of orientations
                 23 % for a perfect orthogonal grid.
                 24 %
                 25 % Behaviour is similar to DELAUNAY(x,y) that
                 26 % can be used to triangulate a mesh into triangles.
                 27 % 
                 28 % It returns an array of indices into the (x,y) matrices
                 29 % where the first dimension walks the quadrangles and 
                 30 % the second dimension are the 4 1D indices into the 
                 31 % (x,y) matrices. Like the results of DELAUNAY where
                 32 % the first dimension walks the triangles and the second 
                 33 % dimension contains the 3 1D indices into the (x,y) matrices.
                 34 %
                 35 % The output is a struct with fields
                 36 % - tri
                 37 % - quat
                 38 % - tri_per_quad
                 39 % - quad_per_tri
                 40 %
                 41 % Example:
                 42 %                                                          
                 43 %     Array of corner points measures 3 x 2.               
                 44 %     Array of center points measures 2 x 1.               
                 45 %                                                          
                 46 %     o-------o-------o.....> 1st matlab dimension         
                 47 %     |1      |2      |3                                   
                 48 %     |   A   |   B   |                                    
                 49 %     |       |       |                                    
                 50 %     o-------o-------o                                    
                 51 %     .4       5       6 indices when treating 2D array as 1D array
                 52 %     .                  walking 1st dimension first.
                 53 %     .                                                    
                 54 %     v                                                    
                 55 %                                                          
                 56 %     2nd matlab dimension                                 
                 57 %                                                          
                 58 %     Quadrangle A is spanned by corner indices [1 2 4 5]  
                 59 %     Quadrangle B is spanned by corner indices [2 3 5 6]  
                 60 %                                                          
                 61 %     Note that the number of quadrangles is equal         
                 62 %     to the number of cell centers.                       
                 63 %                
                 64 % Example: D = triquat([1 2 3;4 5 6;7 8 9],[1 4 7;2 5 8;3 6 9])
                 65 % 
                 66 % see also: QUAT, TRI2QUAT, DELAUNAY, GRADIENT2
