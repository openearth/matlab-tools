                  8 %  --------------------------------------------------
                  9 %
                 10 %  out = addrowcol(in,marker) adds a row and column
                 11 %  filled with the value of marker.
                 12 %
                 13 %  out = addrowcol(in,dm,dn) adds dm rows and 
                 14 %  dm columns with nans.
                 15 %
                 16 %  out = addrowcol(in,dm,dn,marker) adds dm rows and 
                 17 %  dm columns with the value of marker.
                 18 %
                 19 %  If either dm and/or dn is negative, existing 
                 20 %  rows/columns are shifted by |dn| or |dm|
                 21 %  and the first |dn| or |dm| columns are given the 
                 22 %  value nan (default) or marker.
                 23 %
                 24 %  Example:
                 25 %     addrowcol(ones(2), 1, 1,Inf) yields
                 26 %     
                 27 %          1     1   Inf
                 28 %          1     1   Inf
                 29 %        Inf   Inf   Inf
                 30 %     
                 31 %     addrowcol(ones(2),-1,-1,Inf)  yields
                 32 %     
                 33 %        Inf   Inf   Inf
                 34 %        Inf     1     1
                 35 %        Inf     1     1
                 36 %
                 37 % Works also for adding 1D real or character arrays. 
                 38 % The character array concatanation can be useful when 
                 39 % preparing an char array for a vectorized text(...) call.
                 40 %
                 41 % * NOTE that when adding an 1D array, dm or dn should be 1.
                 42 %   as the array is added character by character 
                 43 %   so when adding 'ab' with dm =2, then 'aabb' is added
                 44 %   and not 'abab' as expected. Example:
                 45 %
                 46 %   >> addrowcol('a',0,2,'bc')
                 47 %   
                 48 %   ans =
                 49 %   
                 50 %   abbcc
                 51 %
                 52 % * NOTE that the first element of boerder is added
                 53 %   close to the original array. Example:

                 56 %

                 62 %   
                 63 %   ans =
                 64 %   
                 65 %   cba
                 66 %
