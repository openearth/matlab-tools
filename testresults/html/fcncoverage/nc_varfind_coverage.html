                  6 %   Finds the variable name in a netCDF file where the specified attribute
                  7 %   name (e.g. 'standard_name') matches with the specified attribute value
                  8 %   (e.g. 'time'). The function returns a string with the variable name if
                  9 %   an attributename and -value match was detected, an empty variable if
                 10 %   no match was found, and a cell array when more then 1 variable match description.
                 11 %
                 12 %   ncfile  = name of local file, OPeNDAP address, or result of ncfile = nc_info()
                 13 %   varname = string variable containing 
                 14 %             the variable name where an attribute name and value match should be detected.
                 15 %
                 16 %   The following <keyword,value> pairs have been implemented accepted (values indicated are the current default settings):

                 18 %       'attributevalue', []       = attributevalue to search for in netCDF file
                 19 %
