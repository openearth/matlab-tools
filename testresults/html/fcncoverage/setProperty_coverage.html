                  3 %
                  4 % Routine to set properties based on PropertyName-PropertyValue 
                  5 % pairs (aka <keyword,value> pairs). Can be used in any function

                 12 %
                 13 % input:
                 14 % OPT      = structure in which fieldnames are the keywords and the values are the defaults 
                 15 % varargin = series of PropertyName-PropertyValue pairs to set
                 16 % OPT2     = is a structure with the same fields as OPT. 
                 17 %
                 18 %            Internally setProperty translates OPT2 into a set of
                 19 %            PropertyName-PropertyValue pairs (see example below) as in:
                 20 %            OPT2    = struct( 'propertyName1', 1,...

                 23 %            OPT     = setProperty(OPT, varcell{:});
                 24 %
                 25 % output:
                 26 % OPT     = structure, similar to the input argument OPT, with possibly
                 27 %           changed values in the fields
                 28 % Set     = structure, similar to OPT, values are true where OPT has been

                 30 % Default = structure, similar to OPT, values are true where the values of
                 31 %           OPT are equal to the original OPT

                 33 % Example:
                 34 %
