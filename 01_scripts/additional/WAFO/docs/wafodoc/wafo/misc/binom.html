binom

PURPOSE

calculates the binomial coefficient n!/((n-m)!*m!)

SYNOPSIS

y=binom(n,m)

DESCRIPTION 

  BINOM calculates the binomial coefficient n!/((n-m)!*m!)
 
  CALL:  y=binom(n,m);
 
  Example:%  
     binom(5,2)    % Should be 10.
