sinc

PURPOSE

Sin(pi*x)/(pi*x) function.

SYNOPSIS

y = sinc(x)

DESCRIPTION 

 SINC Sin(pi*x)/(pi*x) function. 
  
   CALL:  y = sinc(x); 
  
     y = sin(pi*x)/(pi*x)    if x ~= 0 
       = 1                   if x == 0 
  
  Example:  
   x =linspace(-5,5)';  
   plot(x,sinc(x)) 
  
  See also  sin
