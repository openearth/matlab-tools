wexprnd

PURPOSE

Random matrices from an Exponential distribution

SYNOPSIS

R = wexprnd(m0,varargin);

DESCRIPTION 

 WEXPRND Random matrices from an Exponential distribution
  
  CALL:  R = wexprnd(m0,sz)
      
         R = matrix of random numbers
        m0 = mean of the Exponential distribution, m0>0
        sz = size(R)    (Default size(m0))
             sz can be a comma separated list or a vector 
             giving the size of R (see zeros for options).
 
  The random numbers are generated by the inverse method. 
 
  Example:
    R=wexprnd(5,1,100);
    phat=wweibplot(R)     % Exp is Weibull with c=1
 
  See also  wexpinv, wweibcdf, wweibplot
