wgaminv

PURPOSE

Inverse of the Gamma distribution function

SYNOPSIS

x = wgaminv(F,a,b)

DESCRIPTION 

 WGAMINV Inverse of the Gamma distribution function
 
  CALL:  x = wgaminv(F,a,b)
 
         x = inverse cdf for the Gamma distribution evaluated at F
         a = parameter, a>0
         b = parameter, b>0 (default b=1)
 
  Example:
    F = linspace(0,1,100);
    x = wgaminv(F,5);
    plot(F,x)
 
  See also  wgamcdf
