wnormpdf

PURPOSE

Normal probability density function

SYNOPSIS

f = wnormpdf(x,m,v);

DESCRIPTION 

 WNORMPDF Normal probability density function 
  
  CALL:  f = wnormpdf(x,m,v); 
  
         f = density function evaluated at x 
         m = mean     (default 0) 
         v = variance (default 1) 
   
  Example:  
    x = linspace(-3,3,200); 
    p1 = wnormpdf(x,0,1); p2 = wnormpdf(x,.5,0.25); 
    plot(x,p1,x,p2)
