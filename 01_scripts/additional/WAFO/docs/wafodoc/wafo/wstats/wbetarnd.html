wbetarnd

PURPOSE

Random matrices from a Beta distribution

SYNOPSIS

R = rbeta(a,b,varargin)

DESCRIPTION 

 WBETARND  Random matrices from a Beta distribution
 
  CALL:  R =wbetarnd(a,b,sz)
 
        R = matrix of random numbers
     a, b = parameters
       sz = size(R)    (Default size(a))
             sz can be a comma separated list or a vector 
             giving the size of R (see zeros for options)
  
   The random numbers are generated by the inverse method. 
  
  Example:
    R=wbetarnd(4,4,40);
    plot(R,'.')
