discar

PURPOSE

Discretizes an AR(1) process.

SYNOPSIS

[P,uu] = discar(u,a1,m,s)

DESCRIPTION 

  DISCAR   Discretizes an AR(1) process. 
      System equation: X(k) = -a1*X(k-1) + e(k),  e(k) = N(m,s^2) 
  
  P = discar(u,a1,s,m) 
  
  P = Transition matrix, 
      P(i,j) = P ( u(j)<X(k)<u(j+1) |  u(i)<X(k-1)<u(i+1) ) 
  
  u  = Discretization levels. 
  a1 = System parameter. 
  m  = Innovation mean. 
  s  = Innovation std.
