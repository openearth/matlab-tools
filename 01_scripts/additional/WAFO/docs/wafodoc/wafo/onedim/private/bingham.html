bingham

PURPOSE

returns the N-point Bingham window in a column vector.

SYNOPSIS

w = bingham(n,p)

DESCRIPTION 

  BINGHAM  returns the N-point Bingham window in a column vector.
 
   CALL: win = bingham(n,p)
         
     win = bingham window
       n = number of points
       p = 0..1 (default 0.2) 
          p = 0 -> rectangular window
          p = 1 -> hanning window
 
  See also hanning, parzen
