recinit

PURPOSE

setup all global variables of the RECDEMO

SYNOPSIS

recinit

DESCRIPTION 

 RECINIT  setup all global variables of the RECDEMO

CROSS-REFERENCE INFORMATION

This function calls:
  • dat2spec
    		•  Estimate one-sided spectral density from data.
  • dat2steep
    		•  Extracts waveheights and steepnesses from data.
  • dist2dfit
    		•  Parameter estimates for DIST2D data.
  • dist2dpdf2
    		•  Joint 2D PDF computed as f(x1|X2=x2)*f(x2)
  • dist2dsmfun2
    		•  Smooths the conditional DIST2D distribution parameters.
  • evalpdf
    		•  evaluates a PDF struct by interpolation
  • findoutliers
    		•  Finds the indices to spurious points in a timeseries
  • geth1line
    		•  Extracts the first comment line (the H1 line) of a m-file
  • gravity
    		•  returns the constant acceleration of gravity
  • kdebin
    		•  Binned Kernel Density Estimator.
  • kdeoptset
    		•  Create or alter KDE OPTIONS structure.
  • qlevels2
    		•  Calculates quantile levels which encloses P% of data
  • reconstruct
    		•  reconstruct the spurious/missing points of timeseries
  • troptset
    		•  Create or alter TRANSFORM OPTIONS structure.
  • waforoot
    		•  Root directory of WAFO installation.
  • cd
    		•  Change current working directory.
  • cell
    		•  Create cell array.
  • clear
    		•  Clear variables and functions from memory.
  • fullfile
    		•  Build full filename from parts.
  • linspace
    		•  Linearly spaced vector.
  • num2str
    		•  Convert number to string. (Fast version)
  • pwd
    		•  Show (print) current working directory.
  • trapz
    		•  Trapezoidal numerical integration.
This function is called by:
  • recfig
    		•  callback implementing functions of RecDemo
