papermenu

PURPOSE

displays a user interface to the paper scripts of WAFO

SYNOPSIS

papermenu(kind)

DESCRIPTION 

  PAPERMENU displays a user interface to the paper scripts of WAFO
 
  CALL:  papermenu(kind)
 
     kind = 0, displays a point and click menu (default)
            1, displays a text driven menu
 
  See also  wafomenu
