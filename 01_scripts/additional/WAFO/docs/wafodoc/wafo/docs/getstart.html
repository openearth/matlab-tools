getstart

PURPOSE

ideas for getting started with WAFO

SYNOPSIS

This is a script file.

DESCRIPTION 

  GETSTART  ideas for getting started with WAFO
 
     0). Install WAFO -- see install.m
 
     1). Get the manual from the home page
            http://www.maths.lth.se/matstat/wafo/
         if you do not already have it in the DOCS directory.
 
     2). Peak into the online documentation files by using wafomenu.m
  
     3). Run some of the scripts in the DEMOS and PAPER directory
 
     4). Look at the help header of a specific function and try out the
         examples if given. 
 
     5). Analyze some of the datasets in WAFO/data
 
     6). If you have come to this point, you are an experienced WAFO
         user. Congratulations.
