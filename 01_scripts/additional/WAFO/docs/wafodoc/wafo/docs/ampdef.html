PURPOSE

wave heights and amplitude definitions and nomenclature

SYNOPSIS

DESCRIPTION 

  AMPDEF wave heights and amplitude definitions and nomenclature
 
  Definition of wave amplitude and wave heights:
 ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
 
             <----- Direction of wave propagation
  
              
            ...............c_..........
            |             /| \         |
        Hd  |           _/ |  \        |  Hu
    M       |          /   |   \       |
   / \      |     M   / Ac |    \_     |     c_  
  F   \     |    / \m/     |      \    |    /  \ 
 ------d----|---u------------------d---|---u----d------ level v
        \   |  /|                   \  |  /      \L
         \_ | / | At                 \_|_/     
           \|/..|                      t
            t
  
   Ac   = crest amplitude
   At   = trough amplitude
   Hd   = wave height as defined for down crossing waves
   Hu   = wave height as defined for up crossing waves
