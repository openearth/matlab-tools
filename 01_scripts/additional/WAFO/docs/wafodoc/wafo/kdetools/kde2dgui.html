kde2dgui

PURPOSE

GUI to Kernel Density Estimator in two dimensions.

SYNOPSIS

varargout = kde2dgui(varargin)

DESCRIPTION 

 KDE2DGUI GUI to Kernel Density Estimator in two dimensions.
 
   CALL:  kde2dgui
 
  Example:
 
   data = wraylrnd(1,1000,2)
   kde2dgui
 
  See also  kde1dgui, kde
